N2PK VNA Introduction and Quick Start Guide

Introduction

This documentation describes the Linux graphical user interface for the open hardware N2PK Vector Network Analyzer (VNA). The N2PK VNA measures scattering parameters of electrical devices under test (DUTs) in the frequency range 50 kHz to 60 Mhz, or other ranges by use of a transverter.

There are several versions of the hardware as well as many variations on the construction of the device. This software works with the following USB based versions of the N2PK VNA:

Vendor ID Product IDs Description
0547 1004, 1005 revision 4: 100mA dual-detector board
0547 1008, 1009 revision 4: 500mA dual-detector board
0547 100a, 100b revision 5: 100mA dual-detector board
0547 100c, 100d revision 5: 500mA dual-detector board

Support can be added for other revisions of the hardware if there's sufficient interest.

The supported operating system is Linux. If you'd like to help port/test the software to other POSIX-like operating systems, e.g. MacOS, BSD, Solaris or POSIX environment of Windows, contact me scott_guthridge@rompromity.net.

The VNA hardware was designed by Paul Kiciak, ham radio call sign N2PK. Documentation on the hardware can be found here: N2PK VNA Hardware. You can order board kits for the VNA and several accessory boards from the same site. The software can be found here: N2PK VNA github.

Quick Start Guide

The following checklist will take you from having just installed the software to making your first DUT measurement.
  1. Add Users to the n2pkvna Group
  2. Start the N2PK VNA app and select the Setup tab
  3. Calibrate the VNA Reference Frequency
  4. Identify the Hardware Configuration
  5. Create the VNA Setup
  6. Select the Calibrate tab
  7. Create the Calibration
  8. Select the Measure tab
  9. Connect the device to be measured
  10. Perform a Scan
  11. Display the Result
  12. Save the result to a File
